# PORTFOLIO_SQL
Hello Reader and Welcome to my SQL Portfolio!

I am a self-taught data enthusiast currently transitioning into the world of SQL data analytics. This repository tracks my progress, showcasing the projects I've built to solve problems using structured data.

My SQL Toolbox:

  Basic Queries: Filtering with WHERE, sorting with ORDER BY, and limiting results.
  Data Aggregation: Summarizing data using GROUP BY, HAVING, and COUNT/SUM/AVG.
  Joining Tables: Combining datasets using JOINs and UNIONs.
  Logic: Using CASE statements to categorize data.
  Organization: Writing readable code with CTEs
  Advanced Data Aggregation: Manipulating data using WINDOW FUNCTIONS: Aggregating, Ranking, Moving Aggregations. 
  Advanced Summary Stats: Summarizing data using CUBING and ROLLUP.
  Advanced Tables: Creating PIVOT TABLES using CROSSTAB
